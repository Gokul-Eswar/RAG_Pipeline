# Technology Stack

## Core Language & Runtime
*   **Python:** Primary programming language for all microservices and data pipelines.

## Data Ingestion & Streaming
*   **Redpanda (Kafka):** High-performance, Kafka-compatible streaming platform for real-time event ingestion.

## Data Processing
*   **Apache Spark:** Distributed processing engine for large-scale data transformation and feature engineering.

## Natural Language Processing (Extraction)
*   **spaCy:** Industrial-strength NLP for entity recognition and linguistic analysis.
*   **Transformers (Hugging Face):** State-of-the-art models for deep semantic extraction and embedding generation.
*   **Sentence-Transformers:** Specialized embeddings for semantic search and vectorization.

## Memory Layer (Databases)
*   **Qdrant / Milvus:** High-performance vector databases for semantic memory and similarity search.
*   **Neo4j:** Native graph database for structural memory and complex relationship mapping.

## API Layer
*   **FastAPI:** Modern, high-performance web framework for building APIs with Python 3.7+ based on standard Python type hints.
*   **Uvicorn:** Lightning-fast ASGI server implementation for FastAPI.

## Orchestration & Frameworks
*   **Airflow / Dagster:** Workflow management platforms to programmatically author, schedule, and monitor pipelines.
*   **LangChain / Llama-Index:** Frameworks for developing applications powered by large language models (LLMs).
*   **Ollama:** Local LLM runtime for executing models like LLaMA 3 and DeepSeek.

## Development & Quality Assurance
*   **Pytest:** Robust testing framework for unit and integration tests.
*   **Black:** Uncompromising Python code formatter.
*   **Flake8:** Tool for style guide enforcement and linting.
